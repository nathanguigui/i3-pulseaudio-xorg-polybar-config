## Prerequisites:

*  use Xorg
*  use i3
*  use polybar


## How to install:

*  add the content of `.config/i3/config` to your i3 config
*  add the content of `.config/polybar/config` to your polybar config
*  copy the script `.config/polybar/sound.sh` to your `~/.config/polybar` folder
